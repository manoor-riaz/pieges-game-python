# Jeu de stratégie – Pièges

## Description
Développement d’un jeu de stratégie inspiré du jeu "Pièges !" (Stay Alive), dans le cadre d’un projet académique.

Le jeu repose sur une grille 7x7 composée de tirettes permettant de créer des trous. Les joueurs doivent déplacer ces tirettes afin de faire tomber les billes adverses et rester le dernier joueur en jeu.

## Fonctionnalités
- Implémentation du moteur du jeu (gestion de la grille et des tirettes)
- Déplacement des tirettes et mise à jour des positions
- Gestion des billes et détection des chutes
- Système de jeu au tour par tour
- Interface en mode terminal
- Gestion des interactions et des règles du jeu

## Technologies
- Python
- Structures de données

## Contexte du projet
Projet réalisé dans le cadre de la Licence Informatique.

Le projet consiste à implémenter un moteur de jeu complet, incluant la gestion de la grille, des tirettes et des interactions entre les éléments, à partir d’un cahier des charges précis.

## Versions du projet
- V1 : version simplifiée en mode solo avec logique de base
- V2 : version complète avec gestion des règles du jeu et interactions

## Résultat
Projet académique validé avec une note de 17/20.

- Modélisation de structures de données
- Gestion d’algorithmes et de logique complexe
- Implémentation de règles de jeu
